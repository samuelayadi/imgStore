This is a movie discovery react app that was built by consuming tmdb api

*******To Open it******
1. Clone the repo to your local machine using `git clone https://github.com/samuelayadi/imgStore.git
2.You change into client directory by using the command "cd directory" 
3. Run "npm install" in order for all dependencies to be installed
4.Run npm start
5.Open the project on your browser 


****Component Structure and Arrangement****
App.jsx is the root Component and it contains Routers to Home, Login, Signup and Gallery components
Home Component contains the Navbar, ImageCardContainer, Footer. 
Login Page has LoginForm components
Signup page has signupForm component
The Gallery page has Navbar, AddFile, ImageCardContainer and Footer components
